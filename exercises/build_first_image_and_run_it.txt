- pick a programming language:
	python
	ruby
	C
	C++
	C#
	Java
- pick a docker image that has your programming langauge at docker hub.
- create a "Dockerfile" which is based on that image.
- create an simple application in the language of your choosing.
- amend the Dockerfile to add your application to the image.
- build the image.
- run the image.
- kill the image.
